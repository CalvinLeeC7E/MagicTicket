## ErrorCode

| Code | Mess                            |
|:----:|---------------------------------|
|  0   | No Error                        |
|  1   | Not found magic ticket          |
|  2   | Token address not match         |
|  3   | Token value less than required  |
|  4   | Mint time greater than required |
